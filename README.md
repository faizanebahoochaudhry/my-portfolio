## 🛠️ Installation & Setup

If you'd like to run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/faizanebahoochaudhry/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.

## 📁 Project Structure

- `/src/app`: Next.js App Router pages and layouts.
- `/src/components`: Reusable React components (UI, Navigation, etc.).
- `/src/styles`: Global and module-specific CSS.
- `/src/data`: Static data configurations for projects and experiences.
- `/public`: Static assets, mockups, and icons.
